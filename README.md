# KROW

Aplicativo mobile de gestão de projetos e tarefas, desenvolvido com React Native + Expo.

## Tecnologias

- Expo SDK 54
- Expo Router
- TypeScript
- React Context API

## Como rodar

```bash
npm install
npx expo start
```

Escaneie o QR Code no Expo Go para abrir no celular.

## Observações

- Sem banco de dados, dados ficam apenas na sessão
- Progresso dos projetos calculado automaticamente pelas tarefas concluídas
